# Sustainable Hydroponic Farming Intelligent Monitoring and Automation
This project presents an IoT-based Hydroponic Farming System developed using ESP32 for intelligent monitoring and automation.

The system continuously monitors:

- pH
- TDS
- Temperature
- Humidity
- Water Level

Sensor data is transmitted to the Blynk IoT platform through Wi-Fi for remote monitoring. Whenever the water level becomes low, the system automatically activates the refill water pump and simultaneously sends push notifications through Blynk along with email alerts to the user.

## Features

- Real-time monitoring
- ESP32 based controller
- Wi-Fi connectivity
- Blynk IoT Dashboard
- OLED Display Output
- Automatic Water Refill
- Air Pump Control
- Push Notifications
- Email Alerts
- Remote Monitoring

- ## Software

- Arduino IDE
- Blynk IoT
- Proteus
- GitHub

- ## Working
1. Sensors collect data.
2. ESP32 processes the readings.
3. Data is uploaded to Blynk using Wi-Fi.
4. OLED displays live values.
5. Water level is continuously monitored.
6. Low water level activates refill pump.
7. User receives Blynk notification.
8. User receives Email Alert.
9. Air pump continuously supplies oxygen.
